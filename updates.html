Game Updates

Version 2.5.0 - Frostbite Update

November 15, 2023

  • Added new Ice Age world with 10 levels
  • Introduced 5 new frost-themed plants
  • Added 4 new zombie types with freezing abilities
  • Improved game performance on Android devices
  • Fixed crash issues on some PC configurations

Version 2.0.0 - Major Content Update

August 3, 2023

  • Added full Android support
  • Introduced new Endless Mode with leaderboards
  • Added 15 new plants and 10 new zombies
  • Implemented cloud save functionality
  • Redesigned UI for better mobile experience

Version 1.5.0 - Bug Fixes

May 20, 2023

  • Fixed memory leak issues on Windows 10/11
  • Balanced plant and zombie stats
  • Improved controller support
  • Added resolution options up to 4K
  • Fixed several progression bugs in Adventure Mode

Version 1.0.0 - Initial Release

February 10, 2023

  • Initial release of Plants vs Zombies Fusion Edition
  • Includes 40 new plants and 25 new zombies
  • 3 new worlds with unique mechanics
  • New game modes: Survival and Puzzle
  • HD graphics support
